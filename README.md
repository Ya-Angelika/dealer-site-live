# АЦ Автосалон — статический сайт (для разработчиков)

Готовый статический сайт. **Сборка не нужна** — просто залейте содержимое этой папки в корень веб-сервера.

## Структура
```
/                     ← заливать ВСЁ содержимое этой папки в корень сайта
├── index.html        ← главная (точка входа)
├── auto.html         ← каталог
├── car.html          ← детальная страница авто
├── credit.html       ← автокредит (калькулятор)
├── tradein.html      ← обмен авто
├── stocks.html       ← акции
├── contacts.html     ← контакты
├── quiz.html         ← подбор авто
├── css/
│   └── style.css     ← все стили (один файл)
├── js/
│   ├── jquery.min.js ← jQuery 3.7.1 (локально)
│   └── main.js       ← интерактив (калькулятор, галерея, FAQ, квиз)
└── images/
    ├── *.jpg         ← hero, акции, калькулятор, trade-in
    └── cars/*.png    ← прозрачные вырезки авто для карточек
```

## Деплой
- **Любой хостинг / nginx / Apache:** скопировать всё содержимое папки в корень (`/var/www/html`, `public_html` и т.п.). Открывается `index.html`.
- **Проверка локально:** двойной клик по `index.html`, или `python -m http.server` из этой папки.

## Технологии
- Обычная вёрстка: HTML + CSS + JS (jQuery). Без сборщиков, npm, фреймворков.
- Все пути относительные. Все ассеты локальные (нет внешних CDN, кроме Google Fonts в `<head>`).
- Адаптив: брейкпоинты 1100px / 720px.

## Правки
- Стили — `css/style.css`. Скрипты — `js/main.js`.
- Контент — прямо в `*.html`.
